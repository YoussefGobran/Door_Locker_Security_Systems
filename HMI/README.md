# HMI_ECU (Human Machine Interface)
```
 frequency 8Mhz.
```

**HMI_ECU** is just responsible interaction with the user just take inputs through keypad and display messages on the LCD.

 Using hardware as 2x16 LCD and 4x4 keypad.
